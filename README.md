
# To Do List Api

## (:construction: Em construção)

Modelo de API To Do list desenvolvido junto a Turma 8 da Resilia Educação no módulo 4 do curso de Desenvolvimento Web.


## Rotas

### Usuário
* GET   `/api/usuarios`
* GET   `/api/usuario/:id`
* POST  `/api/usuario`

### Tarefa
* GET   `/api/tarefas`
* GET   `/api/usuario/:id`
* POST  `/api/tarefa`
